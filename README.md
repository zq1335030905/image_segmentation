# image_segmentation
An image segmentation by K-means cluster algorithm with c++. K-means cluster centers update according to two elements: 1.color, 2.coordinate, 3.texture. You can adjust the weight of them in the code.

Env:
VS2015
OpenCV3.1.0
Debug:x64
